# MusicPilot

Generate customized playlists for every mood with your own library, all locally.

### Project Structure

1. `pwa/`
   Contains all the code for the PWA and frontend.

2. `engine/`
   Contains all the backend flask code, basically where the heavy lifting is happeneing.

3. `app.py`
   Basically makes the two parts above work together.
